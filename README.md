# 🎓 E-Learning Platform Frontend

🚀 A styled and interactive multi-page **E-Learning Platform UI** built as part of **CODTECH Internship Task-4**.

---

## 📌 Project Overview

This project is a front-end design of an E-Learning platform that includes:

- 📚 Course Listing Page
- 🎥 Video Learning (Embedded Player)
- 📊 Progress Tracking System
- 🔗 Multi-page Navigation

The goal was to create a clean, user-friendly interface that simulates a real-world online learning platform.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Responsive Design Principles
